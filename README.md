# Middle_Man_Has_Secret
A technical breakdown and proof-of-concept for Man-in-the-Middle (MitM) interception techniques and mitigation strategies through a self made App Called ARPx


## Description
ArpX is a Python-based network security tool designed to perform ARP scanning, host discovery, and simulate Man-in-the-Middle (MITM) attacks for educational purposes.

## Features
- ARP-based network scanning
- Host discovery in local networks
- MITM attack simulation
# ArpX - Network Security & ARP Analysis Tool
- Basic ARP spoofing detection

## Technologies Used
- Python
- Scapy
- Kali Linux

## Learning Outcome
This project helped in understanding:
- ARP protocol
- Packet crafting
- Network vulnerabilities
- Ethical hacking concepts
